

	adb 指令需要安装android SDK并加入环境变量
	
	adb 连接设备, 需要进开发者模式并打开usb debug功能

	adb devices 查看所有连接的设备
	adb root 取得root权限
	adb shell 启动控制终端
	adb install *.apk 安装apk
	adb disable-verity 关闭设备验证机制
	adb remount 取得部分系统敏感目录的写权限
	adb push 源文件或目录 目标文件或目录    将文件烧入到设备
	adb pull 源文件或目录 目标文件或目录    将文件从设备复制出来
	adb setenforce 0  关闭系统权限
	adb shell input keyevent 3  home键
	adb shell input keyevent 4  返回键
	adb shell input keyevent 24 音量加
	adb shell input keyevent 25 音量减
	adb shell input keyevent 26 电源键
