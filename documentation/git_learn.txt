
	git source code https://github.com/git/git

	git clone git@github.com:ahalfbird/self_project_dujun.git
		从远程拉代码
		拉完代码后会在当前目录生成项目的所有文件和.git  .gitignore
		.git 有branchs (保存建的分支) hooks () config (配置文件包含url fetch remote merge等信息) description (仓库的描述信息) info/exclude (排除git管理的文件) 
		HEAD (头节点) hooks (包含了一些push rebase update等操作会执行到的脚本)

	git branch -a 查看所有分支, 一般情况下拉完代码会自动处于主分支
	
	git checkout -b 分支名 origin/分支名 可以创建新的分支
	
	git clean -df 路径 清除本地库所有不属于项目的文件
	
	git reset --hard 提交点 硬复位到提交点, 会将本地所有修改都覆盖, --soft(或者不加)不回退本地已经修改的文件
	
	git add 文件或路径 添加要提交的文件到暂存区
	
	git commit -m "提交信息" 提交修改的文件
	
	git push origin MESSY1 将本地修改合入到远程库
